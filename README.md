# Gesture Controlled Mouse with Voice Assistant

This project implements a Gesture Controlled Mouse with an integrated Voice Assistant. It uses MediaPipe for hand tracking and gesture recognition, allowing you to control your computer's mouse pointer and perform clicks using hand gestures. The included Voice Assistant can launch the gesture controller and perform other basic tasks.

## Features
- **Gesture Control**: Control mouse movement, left/right clicks, double clicks, and scrolling using hand gestures.
- **Voice Assistant**: Launch and manage the gesture recognition via voice commands.
